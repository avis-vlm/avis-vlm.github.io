# AVIS project page

This repository contains the static project page for:

**AVIS: Adaptive Test-Time Scaling for Vision-Language Models**

The page is intended to be hosted at `https://avis-vlm.github.io/`.

## Local preview

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

Create or update the GitHub Pages repository named `avis-vlm.github.io`, then push the contents of this folder to the `main` branch.
